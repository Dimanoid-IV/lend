Политика конфиденциальности

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сайта Webstart.ee, который принадлежит WebStart веб-студии.

Используя сайт, пользователь выражает согласие с данной Политикой конфиденциальности.

1. Общие положения

1.1. Администрация сайта соблюдает требования законодательства в области защиты персональных данных и принимает все необходимые меры для их сохранности.

1.2. Политика применяется ко всей информации, которую сайт может получить о пользователе во время его использования.

1.3. Администрация сайта не проверяет достоверность предоставляемых пользователем персональных данных.

2. Персональные данные пользователей

Сайт может собирать следующие данные:

  • имя, фамилия;
  • адрес электронной почты;
  • номер телефона;
  • данные, указанные в формах обратной связи;
  • техническая информация (IP-адрес, cookies, тип браузера, данные об устройстве).

3. Цели сбора персональных данных

Персональные данные пользователей используются для:

  • обратной связи с пользователем;
  • обработки заявок и запросов;
  • предоставления услуг веб-студии;
  • улучшения качества работы сайта;
  • аналитики и статистики;
  • выполнения требований законодательства.

4. Обработка и хранение данных

4.1. Обработка персональных данных осуществляется законными способами.

4.2. Персональные данные хранятся не дольше, чем это необходимо для целей их обработки.

4.3. Данные не передаются третьим лицам, за исключением случаев, предусмотренных законом или необходимых для оказания услуг (например, сервисы аналитики, хостинг).

5. Использование файлов cookie

Сайт использует файлы cookie для:

  • корректной работы сайта;
  • анализа поведения пользователей;
  • улучшения пользовательского опыта.

Пользователь может отключить cookie в настройках браузера.

6. Защита персональных данных

Администрация сайта применяет организационные и технические меры для защиты персональных данных от:

  • несанкционированного доступа;
  • утраты;
  • изменения;
  • распространения.

7. Права пользователя

Пользователь имеет право:

  • получать информацию о своих персональных данных;
  • требовать их уточнения, блокирования или удаления;
  • отзывать согласие на обработку персональных данных, обратившись по контактам сайта.
